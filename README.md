# 📚 Online Book Store

A clean and responsive Book Store website designed to provide an engaging browsing experience.  
Users can explore books, discover new arrivals, and read reviews through a simple and attractive interface.

---

## 🚀 Highlights

- 📖 Organized book sections for easy navigation  
- 🌟 Featured collection with pricing details  
- 🆕 Latest arrivals section  
- 💬 User review cards with ratings  
- 📝 Blog section for reading insights  
- 📩 Newsletter subscription layout  
- 📱 Responsive design for all devices  

---

## 🛠️ Built With

- HTML  
- CSS  
- Font Awesome Icons
